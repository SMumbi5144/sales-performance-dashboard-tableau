# Sales Performance Dashboard - Tableau

##Project Overview
An interactive sales and customer dashboard built in Tableau Public,analyzing year-over-year sales performance across products and regions.

##Tools Used
-Tableau Public
- Excel / CSV (dataset)

 ##Data Source
 Data provided by [Data with Baraa](https://www.datawithbaraa.com)

 ##Dashboards Built
 -Sales Dashboard - KPIs,monthly trends,product subcategory comparison
 -Customer Dashboard - customer segments,top 10 customers by profit
